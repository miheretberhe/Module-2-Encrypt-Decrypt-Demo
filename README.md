# Module-2-Encrypt-Decrypt-Demo
Two types of encryptions using Node.js
1. Symmetric Encryption.
    It uses one shared secret key.
    The script generates a random key and IV.
    It encrypts a message and then decrypts it back.
2. Asymmetricn Encryption
     Uses a public key to encrypt and a private key to decrypt
     The script generates a keypair.
     It encrypts a message with the public key and decrypts it with the private key.
